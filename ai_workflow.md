# AI自律型ビジネス・ワークフロー (Antigravity × Claude × Vercel)

一人で「会社」を動かすための、AIエージェントたちの連携フローです。役割（ロール）を意識して使い分けるのがコツです。

## 1. 役割の使い分け (Role Play)

| エージェント | 役割 | 使いどころ |
| :--- | :--- | :--- |
| **Claude** | **CMO / コピーライター** | ・新商品の名前やキャッチコピーを考える<br>・Instagramの投稿文を作る<br>・ブランドのストーリーをさらに深掘りする |
| **Antigravity** | **CTO / 実装者** | ・Claudeが考えたコピーをサイトに反映する<br>・デザインを修正したり、新しいページを追加する<br>・プログラムのエラーを直し、GitHubへ送信する |
| **Vercel** | **店舗 / プラットフォーム** | ・Antigravityが送ったコードを、即座にWebサイトとして公開する<br>・「いま、お店がどう見えているか」の最終確認場所 |

## 2. 運営の黄金フロー (The Loop)

1.  **【企画】Claude と相談する**
    *   例：「秋の新作『Sunset Drift』という豆を売りたいから、心に刺さる紹介文を書いて」
2.  **【実装】Antigravity へ指示する**
    *   例：「Claudeが書いたこの文を、商品一覧ページに追加して。画像は『夕陽とコーヒー』のイメージでお願い」
3.  **【公開】Vercel で確認する**
    *   Antigravity が作業を終え、GitHubへPush（送信）すると、**Vercel は自動でサイトを更新します。** あなたは Vercel のURLを開いて、更新された「新店舗」を確認するだけです。

## 3. なぜこの組み合わせが最強なのか
*   **Antigravity** は、あなたのパソコンの中（ローカル環境）のファイルを直接触れます。
*   **Vercel** は、GitHub という「クラウド上の倉庫」と直結しています。
*   **GitHub** を仲介役にすることで、あなたが何もしなくても **Antigravity → GitHub → Vercel** という「自動出荷ライン」が出来上がります。

---
このフローに慣れると、あなたは「コードを書く人」ではなく、**「AIに指示を出してブランドの未来を決めるCEO」**として、ビジネスを急速に成長させていくことができます。
